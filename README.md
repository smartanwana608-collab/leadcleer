# CSV Contact Tools

A lightweight, browser-based tool for processing and organizing CSV contact files.

## Features
- Filter real estate agents from contact lists
- Preserve all original CSV columns
- Download clean, separated CSV files

## Philosophy
- No data is uploaded or stored
- Processing happens locally in the browser
- Simple, safe, and transparent workflows

## Roadmap
- Guided prompt-based CSV operations
- Address completion tools
- Duplicate detection and cleanup

Built for operational efficiency.
